以下是部分FreeCodeCamp习题

Random Poem Machine

用户可以通过点击 new poem 按钮来随机切换诗词，页面整体风格也随之改变，应用了Bootstrap, HTML, CSS, jQuery 和 Wilddog API。

Wikipedia Viewer

用户可以在一个搜索框搜索维基百科的条目，并看到输出结果，也可以点击一个按钮就看到一个随机的维基百科条目，应用了HTML, CSS, jQuery 和 Wikipedia API。

Pomodoro Clock

用户可以启动一个 5 分钟的番茄钟, 计时器将在 5 分钟后停止，用户也可以为每个番茄钟自定义时长并且重置番茄钟的状态以便启动下一次计时，应用了HTML, CSS 和 jQuery。

JavaScript Calculator

用户可以对两个数字进行加、减、乘、除的运算，可以使用清除按钮清空当前的所有输入内容并且把多个运算连接起来操作，直到按下等号键，计算器输出正确运算结果。应用了HTML, CSS 和 jQuery。
